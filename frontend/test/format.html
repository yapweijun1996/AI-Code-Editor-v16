Use Prettier via backend to format a file on disk. Works best with files within the served project (e.g., frontend/...).
Notes:
  • Static GET can only load files exposed by the server (e.g., under frontend/). Backend formatting can target any project file.
  • If the file is not served statically, use your editor to inspect the formatted result on disk.
